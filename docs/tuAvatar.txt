tuAvatar.vue
Props
badgePosition prop: the position of the prop which are i) top-right ii) top-left iii) bottom-left
pointer prop: on hover cursor to be as pointer if boolean true else not
circle prop: the avatar to be shaped as circle if boolean true else not
square prop: the avatar to be shaped as sqare if boolean true else not
history prop:the avatar to be given special css similar to a button
loading prop: the loading animation to be shown if boolean true
historyGradient prop: special grading to be given to history style. works only if history prop to be true
size prop: size of the badge possible values are in pixels
text prop: text to be shown in the avatar
